**Pingy** is a lightweight and professional tool to check DNS status and measure ping, packet loss, min/avg/max, jitter, and sample count. The interface is **Persian** with a clean and compact design. 🇮🇷✨

---

## Features 🚀

- **Persian UI** with **Vazir** font 🅿️  
- Compact and professional UI/UX 🎨  
- Displays ping statistics: min, avg, max, packet loss, jitter, and sample count 📊  
- Add or remove custom DNS servers or IP addresses ➕➖  
- Probe multiple servers simultaneously 🔄  
- Standalone `.exe` for Windows — no Python required 💻  

---

## How to Use ⚡

1. Download the latest `Pingy.exe` from the [Releases](https://github.com/yourusername/Pingy/releases) page ⬇️  
2. Place the exe file on your computer 🗂️  
3. Run `Pingy.exe` directly ▶️  
4. Add your DNS servers or IPs and start pinging 🌐  

> Note: If the font is not embedded, make sure `Vazir.ttf` is in the same folder as the exe 📝
